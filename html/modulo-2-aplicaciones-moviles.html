Aplicaciones móviles para aprender Front-End

Recursos digitales para estudiar desarrollo web de forma flexible y progresiva

Presentación del módulo

Las aplicaciones móviles de aprendizaje ofrecen acceso rápido a contenidos, ejercicios y ejemplos prácticos desde cualquier lugar. En este módulo se presentan cuatro recursos que pueden apoyar el estudio del desarrollo web Front-End de forma flexible.

Aplicaciones destacadas

1

HTML & CSS Learn

Aprendizaje básico de estructura y diseño web.

2

Web Development Tutorials

Guías paso a paso para aprender desarrollo web.

3

W3Schools

Práctica interactiva con código en tiempo real.

4

Enki

Micro-lecciones para aprendizaje progresivo.

Importancia

Estas aplicaciones fortalecen el aprendizaje autónomo y permiten estudiar desde cualquier lugar.

Reflexión

Compara cada aplicación según su utilidad y tipo de aprendizaje.
