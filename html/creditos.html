Descripción del proyecto

Este material virtual de aprendizaje fue desarrollado como parte de un proyecto académico, con el objetivo de integrar contenidos teóricos y recursos interactivos sobre aplicaciones móviles utilizadas en el desarrollo web Front-End.

Su diseño busca promover un aprendizaje dinámico, autónomo y accesible mediante el uso de herramientas digitales dentro de un entorno educativo.
